For a Mobile User

For a mobile user, Notum is a tool which provide a centralized view of various forms of messages sent by businesses in a standardized alert format. This simplifies user interactions with messages from various vendors.

For Business

For a business, Notum platform provides centralized scheduling and messaging system for communicating with their customers in an efficient way. The businees user interface is currently a web UI and hosted at www.notum.life.
